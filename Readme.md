# Landing Page para Chácara de Eventos

Este projeto é uma landing page desenvolvida para a divulgação de uma chácara que aluga para eventos. Foi construído utilizando HTML, CSS e JavaScript.

## Visão Geral

A landing page tem como objetivo apresentar a chácara e suas instalações, fornecer informações sobre os serviços oferecidos e incentivar os visitantes a entrar em contato para reservas de eventos.

## Tecnologias Utilizadas

- HTML: Utilizado para estruturar o conteúdo da página.
- CSS: Utilizado para estilizar e dar uma aparência atraente à página.
- JavaScript: Utilizado para adicionar interatividade e funcionalidades dinâmicas à página.

## Estrutura do Projeto

O projeto está estruturado da seguinte forma:

- `index.html`: O arquivo HTML principal que contém a estrutura da página.
- `styles.css`: O arquivo CSS que contém estilos para personalizar a aparência da página.
- `script.js`: O arquivo JavaScript que adiciona interatividade à página.
- `Img`: Pasta no qual se encontra as imagens que utilizei no projeto.
